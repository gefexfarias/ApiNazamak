# Instruções Gerais para Agentes - Projeto Nazamak

Siga rigorosamente estas regras ao operar neste repositório:

1. **Idioma de Saída:** Todas as mensagens de commit, descrições de Pull Requests e comentários em código devem ser em **Português (pt-br)**.
2. **Geração de Commits:** Ao ser solicitado a gerar uma mensagem de commit (botão 'Generate' ou similar), utilize o tom profissional em português.
3. **Escopo:** O projeto lida com integração de banco de dados Access e Dashboard React.
4. **Terminologia:** Use os termos de negócio aprovados (Etiqueta, Locação, Saldo, Nota Fiscal).
